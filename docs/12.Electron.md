Electron 使用 Chromium 和 Node.js 来创建跨平台桌面应用。

[Electron Fiddle](https://electronjs.org/fiddle)是运行Electron的工具，可以用来做实验环境。


## Quick Start

首先，创建一个 NodeJS 项目：

    mkdir e1 && cd e1
    npm init

要求：

- 入口文件为 main.js
- author, description 要有内容

然后，安装 Electron：

    npm i -D electron
